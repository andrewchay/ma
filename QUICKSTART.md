# Flow 复楼 MA Agent - 快速启动指南

> 5分钟上手 Flow 复楼智能社交营销AI代理

---

## 🚀 5分钟上手

### Step 1: 激活Agent

在Kimi Code CLI中输入：

```bash
"激活 Flow 复楼营销代理，帮我制定一个社交营销方案"
```

---

### Step 2: 提供Brief

告诉Agent你的需求：

```bash
"我是一家[美妆/3C/快消]品牌，预算[XX万]，
想在[小红书/抖音/微博]进行[新品上市/品牌推广/种草营销]，
目标是[曝光XX万/互动XX万/转化XX]。"
```

---

### Step 3: 选择执行模块

MA Agent会自动调用合适的模块，你也可以指定：

```bash
"使用 StrategyIQ 生成策略"
"调用 MatchAI 筛选KOL"
"启动 ConnectBot 建联"
"使用 CreativePilot 审核内容"
```

---

## 💡 常用指令速查

### 策略生成 (StrategyIQ)

| 指令 | 说明 |
|------|------|
| `生成营销方案` | 基于brief生成完整方案 |
| `分析竞品策略` | 分析竞品的社交营销策略 |
| `平台选择建议` | 推荐适合的平台组合 |
| `预算分配方案` | 优化预算分配建议 |

### KOL匹配 (MatchAI)

| 指令 | 说明 |
|------|------|
| `筛选KOL` | 按条件筛选KOL |
| `分析KOL数据` | 分析KOL数据质量 |
| `组合推荐` | 推荐KOL组合策略 |
| `风险评估` | 评估KOL合作风险 |

### 智能建联 (ConnectBot)

| 指令 | 说明 |
|------|------|
| `生成邀约话术` | 生成个性化建联话术 |
| `自动跟进` | 自动跟进KOL响应 |
| `谈判建议` | 提供报价谈判建议 |
| `合同审核` | 审核合同条款 |

### 创意指导 (CreativePilot)

| 指令 | 说明 |
|------|------|
| `生成创意Brief` | 生成KOL内容Brief |
| `审核内容` | 审核内容合规性 |
| `脚本建议` | 提供视频脚本建议 |
| `优化建议` | 优化内容表现 |

## 📝 完整示例

### 示例1: 新品上市全流程

**输入:**
```bash
我是某美妆品牌，预算80万，要在小红书做新品口红上市，
目标是获得3000万曝光和5万互动。请帮我：
1. 制定整体策略
2. 推荐KOL组合
3. 生成建联话术
4. 提供内容指导
```

**MA Agent会执行:**
1. 调用 **StrategyIQ** 分析需求，确定平台策略和内容方向
2. 调用 **MatchAI** 筛选匹配的美妆KOL（头部+腰部+KOC组合）
3. 调用 **ConnectBot** 生成个性化邀约话术
4. 调用 **CreativePilot** 生成内容brief和参考样稿

---

### 示例2: KOL筛选专项

**输入:**
```bash
帮我筛选适合3C数码品牌的抖音KOL：
- 粉丝量：50-200万
- 互动率：>4%
- 内容类型：科技测评/数码开箱
- 预算：单个3-8万
- 需要10个KOL
```

**MA Agent会:**
- 调用 **MatchAI** 进行筛选
- 输出带匹配度评分的推荐列表
- 提供每个KOL的推荐理由和风险提示

---

### 示例3: 内容审核

**输入:**
```bash
审核这条KOL提交的脚本，检查：
1. 是否符合我们品牌的年轻化调性
2. 是否有合规风险
3. 内容结构是否优化

[脚本内容...]
```

**MA Agent会:**
- 调用 **CreativePilot** 进行多维度审核
- 输出审核报告和修改建议

## 🔧 故障排除

### 问题: Agent没有正确理解需求

**解决:** 提供更详细的上下文
```bash
"我是[具体品牌名]，主营[具体产品]，目标受众是[人群画像]，
竞品是[竞品名称]，预算[具体数字]，周期[具体时间]..."
```

### 问题: 需要调整策略方向

**解决:** 明确反馈
```bash
"这个方案预算分配不合理，建议将小红书预算提高到50%"
"KOL层级需要调整，减少头部，增加腰部KOL数量"
```

### 问题: 输出太泛不具体

**解决:** 要求具体交付物
```bash
"请输出具体的KOL名单，包含账号链接、粉丝数、报价"
"生成可直接使用的建联邮件模板"
```
