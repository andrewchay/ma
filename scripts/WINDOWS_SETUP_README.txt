Flow 复楼 MA Agent - Windows 版使用说明
========================================

1. 解压 ma-agent-windows.zip 到任意文件夹
2. 双击运行 setup.bat
3. 按提示输入你的 API Key（至少需要一个 LLM key 和 Metaso key）
4. 安装完成后，双击桌面的 "MA-Agent" 快捷方式即可启动
5. 启动后，用浏览器打开 http://localhost:8501 使用系统

默认账号密码：admin / admin

支持的 API Key：
- LLM: KIMI_API_KEY / OPENAI_API_KEY / ANTHROPIC_API_KEY / DEEPSEEK_API_KEY
- 搜索: METASO_API_KEY（秘塔，中文搜索推荐）/ TAVILY_API_KEY（可选）

有问题请联系开发者。
